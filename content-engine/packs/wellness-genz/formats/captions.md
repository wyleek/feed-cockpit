# Format: platform captions — Wellness × Gen Z

One natively-written caption per platform. Never paste the same text across platforms.

### TikTok
- 1–2 punchy lines + the source named in-text ("Source: JAMA, June 2026").
- Link handling: TikTok captions don't support links — "study linked in bio" or pinned comment with the source.
- Hashtags: 3–5, niche-first (#sleepscience over #fyp-bait). In caption.

### Instagram Reels
- 2 short paragraphs, line-broken for air. First line must stand alone (it's the preview).
- Source: named in-text + "link in bio". Optionally first comment with full citation.
- Hashtags: 3–8 in caption or first comment; niche > broad.

### YouTube Shorts
- 1–2 lines + full source URL directly in the description (YouTube allows links — use them).
- Hashtags: 2–3 max (#shorts + topic).

### X
- The strongest single claim + source link in the same post. No hashtag stuffing — 0–1 max.
- Thread option: post 2 = the caveat/nuance, post 3 = the source citation in full.

## Universal rules
- Caption restates the receipt even where the link lives elsewhere.
- Caption adds something beyond the script: the caveat, the extra detail, or a genuine question to the comments (comment-bait questions this audience can smell — only ask things we actually want answers to).
