# Gold-standard examples

Put 2–5 APPROVED scripts here, one file each (`example-01.md`, ...). These are the strongest voice signal in the whole pack — the engine imitates these more than it reasons from rules.

Rules for what goes here:
- Only scripts the creator has approved or actually recorded ("would I say this verbatim?" = yes)
- Full package format (hook, script with delivery marks, receipt beat) so structure is modeled too
- When the learning loop produces a heavily-user-edited final the creator loves, ADD IT here and consider retiring the weakest existing example. Keep the set small and elite.
